##### Authentication Required This endpoint requires an OAuth2 Bearer token with the `application_read` scope and a `Deriv-App-ID` header. ## Query Parameters | Parameter | Type | Required | Description | | --- | --- | --- | --- | | `date_from` | string (date) | Yes | Interval start date (YYYY-MM-DD), inclusive, UTC. Example: `2026-05-01` | | `date_to` | string (date) | Yes | Interval end date (YYYY-MM-DD), inclusive, UTC. Example: `2026-05-06` | ## Request Schema #### `Deriv-App-ID` — Required `string` Application identifier used for Personal Access Token (PAT) authentication. Passed as a request header. #### `date_from` — Required `string` Interval start date (YYYY-MM-DD), inclusive, UTC. Pattern: `^\d{4}-\d{2}-\d{2}$` #### `date_to` — Required `string` Interval end date (YYYY-MM-DD), inclusive, UTC. Pattern: `^\d{4}-\d{2}-\d{2}$` ## Response Schema #### `data` — Required `object` Markup and trading aggregates for the caller's applications over the requested date range. Root fields sum across all included apps; `breakdown` lists the same metrics per application. **Object Properties:** ##### `total_app_markup_usd` — Required `number` Sum of application markup attributed to all included apps in the interval. ##### `total_volume_usd` — Required `number` Sum of contract volume (buy price in USD terms) across all included apps in the interval. ##### `total_payout_usd` — Required `number` Sum of all potential payouts (in USD terms) across all included apps in the interval. ##### `total_contract_count` — Required `integer` Total number of contracts counted toward the aggregates across all included apps. ##### `total_client_count` — Required `integer` Number of distinct client accounts that traded under any included app in the interval. ##### `breakdown` — Required `array` Per-application rows; each entry corresponds to one app_id with metrics for that app only. **Array Items:** ###### `app_id` — Required `string` Third-party application identifier. ###### `app_markup_usd` — Required `number` Sum of application markup attributed to this app in USD over the requested interval. ###### `volume_usd` — Required `number` Sum of contract volume (buy price in USD terms) for this app over the requested interval. ###### `payout_usd` — Required `number` Sum of all potential payouts in USD terms for this app over the requested interval. ###### `contract_count` — Required `integer` Number of contracts included in the aggregates for this app. ###### `client_count` — Required `integer` Number of distinct client accounts that traded under this app in the interval. #### `meta` — Required `object` Response metadata. **Object Properties:** ##### `endpoint` — Optional `string` The API endpoint path that was called. ##### `method` — Optional `string` The HTTP method used. ##### `timing` — Optional `integer` Request processing time in milliseconds. ##
